# 🧬 周期共振职业规划系统

基于马江博"政策周期与产业周期"二元分析框架的智能职业规划工具，帮助个人在正确的时间进入正确的行业。

---

## 一、研究背景与意义

### 1.1 问题背景

在当前快速变化的经济环境中，个人职业发展面临着前所未有的挑战：

- **行业轮动加速**：技术革新和政策调整使得行业兴衰周期大大缩短
- **信息不对称**：普通求职者难以获取全面的行业周期信息
- **决策困难**：面对众多行业选择，缺乏科学的决策框架

### 1.2 研究意义

> "企业和个人的财富积累，从来不是单纯努力的结果，而首先是时代浪潮的馈赠。"

本系统将马江博的周期共振理论转化为可操作的职业规划工具，帮助用户：
- 识别当前所处行业的周期阶段
- 判断目标行业的入场时机
- 制定科学的职业转型策略
- 降低职业决策的试错成本

## 二、理论框架支撑

### 2.1 产业周期四阶段

| 阶段 | 特征 | 渗透率 | 职业机会 |
|-----|------|--------|---------|
| 🌱 **初创期** | 技术突破，市场教育 | <5% | 早期机会多，风险高 |
| 🚀 **成长期** | 渗透率快速提升 | 5%-30% | **最佳入场时机** |
| 🏭 **成熟期** | 增速放缓，格局稳定 | 30%-70% | 稳定发展，内卷加剧 |
| ⚠️ **调整衰退期** | 产能过剩，需求萎缩 | >70%或下降 | 风险高，建议转型 |

### 2.2 政策周期四阶段

| 阶段 | 特征 | 政策表现 |
|-----|------|---------|
| 📋 **规划引导期** | 方向明确，扶持出台 | 顶层设计文件 |
| 🎯 **资源聚焦期** | 资金涌入，补贴到位 | 财政补贴、税收优惠 |
| 🔄 **调整退出期** | 优胜劣汰，收紧规范 | 提高准入门槛 |
| 📉 **政策压降期** | 明确限制，产能出清 | 负面清单、限制令 |

### 2.3 四种典型组合（核心决策框架）

```
                    政策周期
          引导期    聚焦期    退出期    压降期
         ┌─────────┬─────────┬─────────┬─────────┐
  初创期 │ 🔴高风险 │ 🟡观察  │ 🔴谨慎  │ 🔴危险  │
产业      │ 押宝期  │ 等待期  │ 观望期  │ 回避期  │
周期     ├─────────┼─────────┼─────────┼─────────┤
  成长期 │ 🟡关注  │ 🟢红利  │ 🟡红利  │ 🟡谨慎  │
         │ 窗口期  │ 交叠期  │ 退坡期  │ 观望期  │
         ├─────────┼─────────┼─────────┼─────────┤
  成熟期 │ 🟡机会  │ 🟡红利  │ 🟡红利  │ 🔴红利  │
         │ 窗口期  │ 交叠期  │ 退坡期  │ 消失期  │
         ├─────────┼─────────┼─────────┼─────────┤
调整衰退│ 🔴回避  │ 🟡观察  │ 🔴红利  │ 🔴红利  │
  期    │         │         │ 退坡期  │ 消失期  │
         └─────────┴─────────┴─────────┴─────────┘
```

**核心结论**：
- 🟢 **红利交叠期**（成长+聚焦）：**最佳入场时机**
- 🔴 **红利消失期**（衰退+压降）：**建议立即转型**

### 2.4 新产业成长期拐点判断7大清单

1. ✅ 技术成本在2-3年内下降50%以上
2. ✅ 龙头企业毛利率超过20%，净利润转正
3. ✅ 政策文件中明确财政资金规模和补贴标准
4. ✅ 市场渗透率在5%-30%之间
5. ✅ 行业资本开支增速维持30%以上
6. ✅ 出现3家以上年营收超过10亿的企业
7. ✅ 产业链上下游配套开始完善

## 三、系统使用流程（5步完成职业规划）

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  ① 📄 简历   │ → │  ② 📊 周期   │ → │  ③ 🤖 AI    │ → │  ④ 🛤️ 路径   │ → │  ⑤ 📡 数据   │
│    诊断      │    │   实验室     │    │   规划官     │    │    推演      │    │   哨兵       │
│  (了解现状)  │    │  (查看全景)  │    │  (获取建议)  │    │  (模拟决策)  │    │  (追踪执行)  │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
```

### 详细使用步骤

#### Step ① 简历诊断（了解现状）
- 进入「简历诊断中心」
- 粘贴您的简历内容或选择所在行业
- AI自动识别您的行业周期阶段
- **红色预警**：如果检测到高风险行业，会提示转型建议

#### Step ② 周期实验室（查看全景）
- 查看近300个细分领域的周期分布
- 使用象限图找到"红利交叠期"行业
- 搜索您感兴趣的行业

#### Step ③ AI协同规划官（获取建议）
- 与AI对话，获取个性化职业规划
- 基于RAG知识库，所有建议有理有据
- 可以询问：转型建议、技能准备、时机判断等

#### Step ④ 职业路径推演（模拟决策）
- 选择推演模式：历史回溯 / 未来推演 / 双轨对比
- 模拟不同选择的发展路径
- 量化风险收益比

#### Step ⑤ 数据哨兵服务（追踪执行）
- 添加目标行业到追踪列表
- 监控7大拐点指标（当前为演示数据）
- 辅助判断入场时机

### 4.2 核心功能模块

| 顺序 | 模块 | 功能 | 核心技术 |
|:---:|------|------|---------|
| ① | **简历诊断中心** | 粘贴简历，AI识别行业风险 | LLM解析 + 风险匹配 |
| ② | **周期实验室** | 可视化呈现近300个细分领域周期 | Plotly象限图、Pandas筛选 |
| ③ | **AI协同规划官** | 基于RAG生成深度职业规划报告 | DeepSeek API + RAG检索 |
| ④ | **职业路径推演** | 模拟不同选择的发展路径 | 时间线模拟、双轨对比 |
| ⑤ | **数据哨兵服务** | 追踪7大拐点指标 | 指标雷达图、趋势分析（演示数据） |

### 4.3 RAG（检索增强生成）架构

```
用户输入 → 行业关键词提取 → CSV知识库检索 → 上下文注入 → DeepSeek生成
                ↓                      ↓
          模糊匹配算法          周期理论规则库
          （行业名称、评价）      （四种组合策略）
```

**核心优势**：
- 所有AI建议都有知识库数据支撑
- 避免大模型的"幻觉"问题
- 实现有理有据的职业规划

## 五、快速部署

### 5.1 本地运行

```bash
# 克隆仓库
git clone <repository-url>
cd cycle_master_system

# 安装依赖
pip install -r requirements.txt

# 运行应用
streamlit run app.py

# 浏览器访问 http://localhost:8501
```

### 5.2 Streamlit Community Cloud（推荐）

1. 将代码推送到 GitHub 仓库
2. 访问 [share.streamlit.io](https://share.streamlit.io)
3. 选择仓库并部署
4. 在 Settings > Secrets 中配置 `DEEPSEEK_API_KEY`（可选）

### 5.3 API Key 配置

系统支持三种方式配置 DeepSeek API Key：
1. **环境变量**：`DEEPSEEK_API_KEY`
2. **Streamlit Secrets**：部署平台设置
3. **侧边栏输入**：运行时手动输入（推荐公开试用）

**使用限制**：每位用户每日限 **20次** AI生成调用

## 七、安全与隐私

### 7.1 隐私保护承诺

- ✅ 本系统**不留存**任何用户个人隐私数据
- ✅ 所有对话分析均**实时调用** DeepSeek 大模型处理
