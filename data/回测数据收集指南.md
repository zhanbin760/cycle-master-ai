### 步骤3：数据收集实操

#### 3.1 获取行业指数历史数据（以申万指数为例）

```python
# 示例代码：使用 akshare 获取免费数据
import akshare as ak

# 获取新能源车指数历史行情
df = ak.index_zh_a_hist(
    symbol="930997",  # 中证新能源汽车指数
    period="daily",
    start_date="20200101",
    end_date="20241231"
)

# 计算区间涨幅
start_price = df.iloc[0]['close']
end_price = df.iloc[-1]['close']
return_rate = (end_price - start_price) / start_price * 100
print(f"区间涨幅: {return_rate:.2f}%")
```

### 步骤4：准确性判定标准

| 判定结果 | 判定标准 | 示例 |
|---------|---------|------|
| **准确** | 预测方向与实际一致，且幅度误差<30% | 预测上涨，实际涨125% |
| **基本准确** | 预测方向与实际一致，幅度误差30-50% | 预测上涨，实际涨40% |
| **偏差** | 预测方向与实际相反，或幅度误差>50% | 预测上涨，实际跌10% |
| **反例** | 预测方向完全相反 | 预测衰退，实际大涨 |

## 四、关键注意事项

### 4.1 避免幸存者偏差
- 不要只收集验证成功的案例
- 必须包含"反例"（如2021年煤炭）
- 建议成功:失败案例比例控制在 7:3

### 4.2 时间窗口选择
- 回测周期建议 1-2 年（太短可能是噪音，太长周期可能转换）
- 避开极端市场事件（如2020年3月疫情暴跌）

### 4.3 证据留存
- 保存马江博原文截图（公众号文章）
- 保存政策文件原文链接
- 记录数据来源和时间
